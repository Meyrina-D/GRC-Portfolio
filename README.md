# Information Security GRC Portfolio

Welcome to my Information Security Governance, Risk, and Compliance (GRC) portfolio.

This repository showcases documentation developed through a self-directed academic project to gain practical experience in implementing an Information Security Management System (ISMS) aligned with **ISO/IEC 27001:2022**.

The portfolio demonstrates my understanding of information security governance, risk management, compliance documentation, and security policy development.
