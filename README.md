Information Security GRC Portfolio

This repository contains documentation developed as part of a self-directed academic project to gain practical experience in Governance, Risk, and Compliance (GRC) and ISO/IEC 27001:2022 implementation.

The project simulates the implementation of an Information Security Management System (ISMS) for a fictional SaaS organization, Cyber Lift LLC, and includes documentation commonly prepared during ISMS implementation.

Disclaimer: This repository is an academic portfolio created for learning purposes. "Cyber Lift LLC" is a fictional organization used to demonstrate practical understanding of ISO/IEC 27001:2022, information security governance, and compliance documentation.
